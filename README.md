# Lepays


<b>contexte<b> : Le journal local “Le Pays” souhaite se moderniser et créer un site internet pour relayer les informations de sa version papier. Monsieur Do le directeur du journal, est très attaché au papier et au confort de lecture que peut fournir ce support. Il souhaite donc que le site soit simple d’utilisation et aussi agréable à la lecture que sa version papier.
